\# 💬 Chat Sentiment Analyzer



This is a simple web application built with \*\*Flask\*\* and \*\*TextBlob\*\* that analyzes the sentiment of user input messages. It classifies messages as \*\*Positive\*\*, \*\*Negative\*\*, or \*\*Neutral\*\* and shows the polarity score.



---



\## 🔧 Tech Stack



\- \*\*Python 3\*\*

\- \*\*Flask\*\* (for web framework)

\- \*\*TextBlob\*\* (for sentiment analysis)

\- \*\*HTML + CSS\*\* (for frontend)



---



\## 🚀 Features



\- User enters any message or comment

\- Backend analyzes the sentiment

\- Outputs:

&nbsp; - Sentiment type: Positive / Negative / Neutral

&nbsp; - Polarity score

\- Clean and responsive UI
